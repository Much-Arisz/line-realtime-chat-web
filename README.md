# line-realtime-chat-web

## Introduction
Line OA Real-time Chat Front-end (Web)
## Requirements
- Node.js 18 or higher
- Docker
## Quick Start
1. Install Library
``` bash
$ npm install
```
2. Run Docker Compose
``` bash
$ docker-compose up -d
```
3. Public URL (NGROK) after run docker-compose
- https://drum-star-stud.ngrok-free.app/ 
4. Add Line OA
